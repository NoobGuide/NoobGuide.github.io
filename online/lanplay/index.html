    LAN Play#


    GIOCARE ONLINE?

    Ma allora è possibile giocare online con la Atmosphère?
    NI.


    Attraverso questa guida sarà possibile connettersi a dei server privati che ospitano partite private ad hoc.


    MAI IN NESSUN MODO SARÀ POSSIBILE CONNETTERSI AI SERVER UFFICIALI.


    Il programma che si andrà ad utilizzare è LAN Play, che permetterà alla console di connettersi ai server privati appena menzionati, il tutto con l'aiuto dell'interfaccia grafica fornita dal programma LANPlayGUI.


    ANCHE SENZA ATMOSPHÈRE

    La procedura può essere seguita anche senza l'utilizzo di Atmosphère per alcuni giochi, seguendo per intero la guida, saltando semplicemente la sezione "Preparazione dei file per la microSD"


    PRIMA DI CONTINUARE

    DATO CHE QUESTA GUIDA PREVEDE DI ANDARE ONLINE CON IL CUSTOM FIRMWARE, SI CONSIGLIA CATEGORICAMENTE DI SEGUIRE PRIMA LA SEZIONE RELATIVA ALLA MODALITÀ INCOGNITO PER EVITARE IL BAN E, SOLO DOPO, CONTINUARE CON LE INDICAZIONI CHE SEGUONO.

    L'UTENTE SARÀ L'UNICO RESPONSABILE DI EVENTUALI BAN.


    1. Prerequisiti#

    • Un programma, a seconda del sistema operativo utilizzato:
      • Windows: Npcap installer (selezionare installed in WinPcap API-compatible mode durante l'installazione);
      • macOS: prima installare Homebrew secondo la guida riportata sul sito, poi in un terminale digitare:
        $ brew install libpcap libuv
      • GNU/Linux: installare i seguenti pacchetti:
        • Debian e derivate: libpcap0.8-dev libuv1-dev
        • Arch, RedHat e derivate: libpcap libuv

    • L'ultima release di switch-lan-play (lan-play-win64.exe);

    • L'ultima release dell'interfaccia grafica lan-play-GUI adatta al sistema operativo (lan-play-GUI-xxxx.x.x.x.rar);

    • L'ultima versione del pacchetto completo (ldn_mitm_vx.x.x.zip).


    Per altre e maggiori informazioni di seguito sono presenti link a diversi server e commmunity con cui organizzarsi in partite private.

    • Una lista di giochi che sono stati testati con LAN Play.
    • A questo link si possono trovare alcuni giochi e come accedere alla modalità online.
    • Risulta anche presente una community discord dove incontrare altra gente che usa i server.


    2. Preparazione dei file per il PC#

    Dopo aver installato il programma adatto al sistema operativo, si passa alla configurazione dei programmi necessari da utilizzare su PC.

    Estrarre il pacchetto di lan-play-GUI e mettere all'interno della stessa cartella, il file relativo a switch-lan-play, per comodità.

    Adesso aprire lan-play-GUI.exe come amministratore.

    Una volta aperto, cliccare su Settings, in alto.

    Specificare la cartella dove si trova switch-lan-play attraverso la prima voce delle impostazioni, Lan Play Location.

    Fatto ciò, selezionare Server List.

    Selezionare adesso Download Official Server List e confermare selezionando I'm okay with that, replace it!.

    Appena finirà il processo, dare l'Ok, anche in basso per chiudere le impostazioni e tornare nella pagina principale del programma, dove saranno presenti dei server ai quali sarà possibile connettersi!

    Selezionare adesso Update in alto e lo status dei server verrà aggiornato.

    NOTA BENE: ci si può anche connettere ad un altro server personalizzato selezionando Add New Server. Il nuovo server apparirà in fondo alla lista. A inizio guida sono presenti link utili relativi ad altri server.

    Selezionare adesso Connect to Server su un server che risulta essere online. Una volta fatto spunterà una finestra di comando su Windows che darà informazioni sull'apertura del server sulla rete locale.

    L'effettivo funzionamento del programma è dato sia dai vari ok presenti accanto alle interface, che indica che il server è attivo su tutte le interfacce del PC, sia dalla scritta pcap loop start.

    Fatto ciò, lasciare tutto aperto per com'è e passare alla configurazione dei file sulla microSD di Nintendo Switch.

    4. Creazione della connessione ad hoc#

    ULTIMO AVVISO

    SEGUIRE LA PROCEDURA PER LA MODALITÀ INCOGNITO PRIMA DI ANDARE OLTRE

    Una volta avviata la console, recarsi nelle Impostazioni di sistema, alla voce Internet, Impostazioni Internet e scorrere in basso fino alla voce Configurazione manuale.

    È consigliato andare a creare una nuova connessione, dato che si andranno a mettere indirizzi IP e DNS specifici per poter accedere al servizio.

    Nel Nome della connessione è consigliato mettere un nome qualsiasi che identifichi univocamente la connessione dedicata al servizio LAN Play, chiamandola magari LAN Play.

    Inserire l'SSID, cioè il nome vero e proprio del modem a cui ci si sta connettendo, la password e andare a impostare le Impostazioni indirizzo IP come Manuali, e inserire:

    • Indirizzo IP: 10.13.xx.yy
    • Subnet mask: 255.255.0.0
    • Gateway: 10.13.37.1

    L'indirizzo IP può essere qualsiasi, nel range tra 10.13.0.1 e 10.13.255.254, TRANNE 10.13.37.1.

    Scendere alla voce Impostazioni DNS, impostarli Manuali e inserire:

    • DNS Primario: 163.172.141.219
    • DNS Secondario: 207.246.121.77

    Una volta impostato tutto come sopra, selezionare Salva.

    Se la switch ed il PC sono connessi alla stessa rete, la prova di connessione andrà a buon fine.

    Si è adesso pronti a giocare online con la propria console!
