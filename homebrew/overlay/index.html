    Tesla Menu - Overlay#

    Tesla non è altro che un menù overlay, che appare in qualsiasi momento, dopo aver premuto una ben determinata configurazione di tasti, impostabile a piacere dall'utente, un po' come quando si accede al menù rapido, tenendo premuto il tasto Home della console.

    Tesla si compone di tre parti:

    • nx-ovlloader: una sorta di nx-hbloader, ma sotto forma di sys-module, che carica i file di overlay;
    • Tesla-Menu: l'equivalente dell'hbmenu, ma sotto forma di overlay;
    • libtesla: libreria utile agli sviluppatori (insieme al Tesla-Template) per sviluppare nuovi overlay.

    Fonte GBAtemp


    1. Prerequisiti#

    • L'ultima versione di nx-ovlloader (nx-ovlloader.zip);
    • L'ultima versione di Tesla Menu (ovlmenu.zip);

    • Overlay a vostra scelta, ovvero file con estensione .ovl, reperibili da:

      1. GBAtemp, citato in precedenza;
      2. Pagina GitHub che non fa altro che raccogliere buona parte degli overlay finora disponibili, come quelli per i cheat, per visualizzare le informazioni durante il gioco, oppure ancora cambiare valore di overclock/underclock al volo.
      3. La sezione Altri tool (overlay) della nostra raccolta di homebrew e tool essenziali per Nintendo Switch.


    2. Procedimento#

    ASSICURARSI PRIMA DI TUTTO DI CANCELLARE, SE PRESENTE, LA CARTELLA:

    <microSD>/atmosphere/contents/010000000007E51

    Dopo aver scaricato i file necessari, estrarre i due file .zip direttamente nella microSD.

    Mettere invece i file .ovl all'interno del percorso:

    <microSD>/switch/.overlays/

    NOTA BENE: dato che la cartella comincia con un punto ("."), alcuni sistemi operativi (macOS, GNU/Linux) potrebbero non visualizzarla se non è stata abilitata la visualizzazione dei file nascosti.

    Il risultato, dovrebbe essere come quello delle immagini che seguono.

    Se inoltre si vuole andare a configurare la combinazione di tasti per fare apparire il menù overlay, bisogna andare a modificare il file config.ini presente all'interno del percorso:

    <microSD>/config/tesla/

    NOTA BENE: essendo Tesla ancora in sviluppo, la cartella ed il file potrebbero ancora non essere presenti all'interno dei file .zip scaricati in precedenza. Basterà andare a creare le cartelle e il file a mano, copiando il contenuto dalla nota sottostante.

    config.ini
    1
2
3
4
5
    [tesla]
; Copy this file in the "/config/tesla/" folder
; Possible key names are: A, B, X, Y, DLEFT, DRIGHT, DUP, DDOWN, LS, RS, MINUS, PLUS, SL, SR, L, R, ZL, ZR
; Separate different keys that should be help simultaneously with a plus ( + ) sign
key_combo = ZL + DUP + ZR

    2.1 Tesla-Menu#

    Inserita la microSD nella console ed accesa, basterà andare ad aprire il gioco d'interesse e premere la combinazione di tasti scelta dall'utente, o quella di default: L + DPAD GIÙ + R3.

    A questo punto, scegliere uno degli overlay scaricati in precedenza.
